JavaScript Basics Tutorial

Welcome to this interactive JavaScript tutorial for beginners. Follow along and try out the examples!

1. Variables

JavaScript allows you to declare variables using let, const, or var.

let name = "John";
console.log(name);

2. Functions

Functions allow you to reuse code. Here's an example of a function that adds two numbers:

function add(a, b) {
    return a + b;
}
console.log(add(3, 5));

3. Conditional Statements

Conditional statements let you control the flow of your program based on conditions.

let age = 18;

if (age >= 18) {
    console.log("You are an adult.");
} else {
    console.log("You are a minor.");
}

4. Loops

Loops allow you to repeat code. Here's an example of a for loop:

for (let i = 0; i < 5; i++) {
    console.log(i);
}

5. DOM Manipulation

With JavaScript, you can interact with and modify the HTML structure of a webpage. Here's an example:

// Change the text of the heading
document.querySelector("h1").textContent = "JavaScript is Awesome!";

6. Events

You can respond to user interactions using event listeners. For example:

// Add a click event to a button
document.querySelector("button").addEventListener("click", () => {
    console.log("Button clicked!");
});
